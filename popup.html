Kneuron 1.1.0 - Keyboard Shortcuts

Global Shortcuts

Enter Save or submit form
Works on any form or editing view
In the Javascript and CSS editors - use Alt+S instead
Esc Close or cancel
Closes modals or cancels current action

Code Editors

Alt + S Save
Save in Javascript and CSS editors

Navigation

Alt + 1-6 Navigate main menu
Quick access to main navigation items on 1st column
3 & 4 are swapped for ergonomy (Pages/Tasks)
Alt + ` (backtick / backquote) Go back
Returns to previous sub-section
Alt + S Search
Sets focus on the Filter box if visible

View Settings Tools

Alt + S Settings/Save
Access view settings menu
Alt + Q, A, Z, or X
Quick access to the first through fourth tools in 2nd column
Will auto-navigate to Settings first if needed

Automatic Features

- Automatically expands hidden togglers in list views
- Makes Description text areas taller in Pages/Settings toolbox
- Reduces long grids (3 groups and 3 records max, with fading effect)
- Hides beginners tooltips, except field infos
