Kneuron 1.2.1 - Keyboard Shortcuts

Global Shortcuts

Enter Save or submit form
Works on any form or editing view
In the Javascript and CSS editors - use Alt+S instead
Esc Close or cancel
Closes modals or cancels current action

Code Editors

Alt + S Save
Save in Javascript and CSS editors

Navigation

Alt + 1-6 Navigate main menu
Quick access to main navigation items on 1st column
3 & 4 are swapped for ergonomy (Pages/Tasks)
Alt + ` (backtick / backquote) Go back
Returns to previous section
Alt + S Search
Sets focus on the filter box, if visible

View Settings Tools (a view must be selected)

Alt + S View's Settings
Access view settings' toolbox
Alt + Q, A, Z, or X
Quick access to the first through fourth sub-tools in 2nd column
Will auto-navigate to settings first if needed

Automatic Features

- Incremental filtering on tables, pages, add field to view and copy/move to page selector
- Reduces grids and lists, with fading effect (3 groups / 3 records max)
- Automatically expands hidden togglers in list views
- Makes Description text area larger in Pages/Settings toolbox
- Hides beginners tooltips, except field infos
