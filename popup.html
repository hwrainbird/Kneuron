Kneuron 1.5.0 - Keyboard Shortcuts

Global Shortcuts

Enter Save or submit form
Works on any form or editing view
In the Javascript and CSS editors - use Alt+S instead
Esc Close or cancel
Closes modals or cancels current action
Alt + M Toggle divier to min/max position
Will move the splitter all the way to the left or to the right

Code Editors

Alt + S Save
Save in Javascript and CSS editors

Navigation

Alt + 1-6 Navigate vertical main menu
Quick access to main navigation items in first column
Some accounts have 5 or 6 menus, depending if you have Flows
Alt + W, E, or R Navigate horizontal tabs
Quick access to tab navigation items at top
Alt + ` (backtick / backquote) Go back
Returns to previous section
Alt + S Search
Sets focus on the filter box, if visible

View Settings Tools (a view must be selected)

Alt + S View's Settings
Access view settings' toolbox
Alt + Q, A, Z, or X
Quick access to the first through fourth sub-tools in 2nd column
Will auto-navigate to settings first if needed

Automatic Features

- Re-orders the left menu pane: Data, Pages, Tasks, Flows, Settings and Data Model
- Incremental filtering on Tables, User Roles, Pages and Copy/Move to Page selector
- Adds the Table, Scene and View IDs
- Reduces grids and lists, with fading effect (3 groups / 3 records max)
- Automatically expands hidden togglers in list views
- Makes Description and Email Message text areas larger in Pages editing toolboxes
- Hides beginners' tooltips, except for field information
- Modifies the Record History page to have sticky headers
